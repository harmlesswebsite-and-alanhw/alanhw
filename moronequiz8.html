Anyway, here's how to photoshop images as promised:

  1. Find an image to photoshop. This may be done by going on Google Maps -> Street View, but can also be any other image.
  2. Open it in Photopea or GIMP, whichever you like better.
  3. Start copy pasting the letters.
  4. If you're using Photopea, flatten it using Layer -> Flatten Image, and then save it as PNG.
  5. If you're using GIMP, export it as PNG.
